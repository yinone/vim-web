## The vimrc config
It's a beautiful and simple vimrc config for front-end development

![](vim.png)

## Installation
get the vimrc

    git clone git@github.com:yinone/web_vimrc.git ~/.web_vimrc
    sh ~/.web_vimrc/install.sh

## Update
    
do a git rebase

    cd ~/.web_vimrc
    git pull -rebase
